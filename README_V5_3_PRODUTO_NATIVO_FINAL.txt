RODRIGUES ENTREGADOR V5.3 — PRODUTO NATIVO FINAL

Esta versão consolida os pontos alinhados:
- Topo usa foto real cadastrada do entregador.
- Status fica no próprio botão.
- Verde = Disponível.
- Preto/cinza = Indisponível.
- Vermelho = Restrição.
- Restrições: bateria abaixo de 10%, sem internet, localização desligada, permissão negada, alertas desativados, full-screen pendente, bateria restrita, cadastro pendente/bloqueado e corrida em andamento.
- Card aguardando corridas com radar animado profissional.
- Rodapé com ícones profissionais: Início, Corridas, Ganhos, Histórico e Conta.
- Tela Conta inclui central organizada por seções: Configurações, Financeiro, Conta, Suporte e Sobre.
- Modo desenvolvedor removido do fluxo normal.
- Ganhos com olho profissional para ocultar valores.
- Mapa interno mantido, limpo, sem cards/camadas por cima, somente botão fullscreen.
- Toque no mapa abre mapa grande.
- Rota atualiza em ciclo de 30 segundos.

Como aplicar:
1. Extraia este ZIP.
2. Copie o conteúdo para C:\RSITE\rodrigues-entregador-nativo.
3. Rode APLICAR_V5_3_PRODUTO_NATIVO_FINAL_WINDOWS.bat.
4. Aguarde o GitHub Actions gerar o APK.

Observação honesta:
O pacote está preparado. O funcionamento final precisa ser validado no APK instalado.
