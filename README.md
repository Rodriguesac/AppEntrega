# Rodrigues Entregador V5.2 Operacional

App Android nativo exclusivo da Rodrigues Açaí e Cia.

## Destaques
- Corrige pós-aceite: ao aceitar, abre a corrida ativa.
- Visual mais limpo e operacional.
- Tema claro/escuro.
- Integração com `ofertas_corrida`.
- Fluxo: oferta > aceitar/rejeitar > coleta > pedido retirado > entrega > finalizar.
- Foreground service e notificação urgente.

Mantenha `app/google-services.json` do projeto Firebase.
