RODRIGUES ENTREGADOR V5.2.3 — RESTAURA MAPA INTERNO

Este pacote deve ser usado para corrigir a V5.2.2.

O que ele faz:
- Restaura o mapa/rota interna no app.
- Mantém a correção dos valores de repasse da V5.2.1.
- Mantém o botão para abrir navegação externa.
- Desfaz a retirada do mapa interno.

Como aplicar:
1. Extraia o ZIP.
2. Copie o conteúdo para:
   C:\RSITE\rodrigues-entregador-nativo
3. Rode:
   APLICAR_V5_2_3_RESTAURA_MAPA_WINDOWS.bat
4. Aguarde o GitHub Actions gerar o APK.

Importante:
Não use mais a V5.2.2 sem mapa interno.
